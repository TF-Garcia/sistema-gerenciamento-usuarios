# sistema-gerenciamento-usuarios
*-- Na pasta PHP o index está como indice.php --*
Este é um sistema em PHP com Javascript também. 
É um sistema para gerenciamento de clientes e serviços prestados destinado à organização interna de estabelcimentos.
Inclui cadastro de serviços prestados, datas, filtros por data do serviço, barras de busca e paginadores.
